This script uses Scapy to capture live network traffic and analyze IP packets. It first displays all available network interfaces so the user can choose one, although in this case the default interface is used. It then starts sniffing network packets and, for each packet that contains IP information, it extracts and prints the source and destination IP addresses. At the same time, each captured packet is stored in a list. Once the sniffing process is stopped, all collected packets are saved into a file named `capture.pcap`, which can later be opened and analyzed using tools like Wireshark.
